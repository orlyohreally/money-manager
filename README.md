# Money Manager

### Requirements

Node.js v10.15.3  
npm 6.4.1

### Installation

Clone project
```
git clone https://github.com/orlyohreally/money-manager.git
```

Install the dependencies

```
npm install
```

Run Angular project

```
ng serve --proxy-config proxy.conf.json
```

The project will be available at http://localhost:4200.
